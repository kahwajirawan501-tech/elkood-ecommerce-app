# elkood-ecommerce-app

A professional E-commerce application developed as a technical task for **EL KOOD Technical Solutions**. The app provides a seamless shopping experience, allowing users to browse products, manage their cart, and maintain their profile.

## 🚀 Features

* **Authentication**: Secure user login and signup functionality.
* **Product Browsing**: Fetches products from [Fake Store API](https://fakestoreapi.com/).
* **Search**: Efficient real-time product search.
* **Shopping Cart**: Add/remove items, update quantities, and save the cart locally.
* **Profile Management**: View user details.
* **Persistent Storage**: User data and cart state are preserved locally using local storage.

## 🛠 Tech Stack

* **Framework**: Flutter.
* **State Management**: BLoC (Cubit) for handling application states.
* **API**: Integrated with Fake Store API.
* **Architecture**: Clean, modular structure for scalability and readability.
* **Responsive Design**: Optimized for different screen sizes using `flutter_screenutil`.
